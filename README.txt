Installation instructions
-------------------------

1. Run 'Setup.exe' 
2. Wait...
3. If you don't have .NET Framework Client Profile and XNA Framework Redistrutable 4.0, prepare to wait a while more as the installer gets these. 
4. The game can be found from the Start menu: RuttoPuput -> Hamsters and Plague
5. Play!
